POÄNG

roller derby scoreboard

POÄNG is a roller derby scoreboard. Accept no substitutes.

QCRG Scoreboard is being retired after three seasons of reliable service, with POÄNG scheduled to take its place in January 2011. Adventure-seekers are encouraged to read about or download the latest development version. The technically inclined can follow –or better yet, join in– the progress of this new open source effort on GitHub. For now, discussion of POÄNG will be carried on in the Google group for QCRG Scoreboard.

If you are downloading POÄNG for prerelease evaluation, please go to the Updates tab in the Preferences/Settings dialog before doing anything else and set your preferred release track to “development.” The code is changing quickly, and this will ensure that you get the most recent updates.
